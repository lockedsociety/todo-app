# Todo

A todo app for daily task.

## Features list

- [ ] Task with title and description.
- [ ] Add task.
- [ ] Delete task.
- [ ] Edit task.
- [ ] Timestamp when created.
- [ ] Timestamp when to be done.
- [ ] Save locally.
- [ ] History.
- [ ] Mark as done.
- [ ] Warn when not done when time reached.
- [ ] Download task as pdf or others.
