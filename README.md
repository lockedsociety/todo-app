# Scheduler

A scheduler for daily tasks.

## Features list

- [x] Task with title and description.
- [x] Add task.
- [x] Delete task.
- [x] Edit task.
- [x] Timestamp when created.
- [x] Timestamp when to be done.
- [x] Save locally.
- [x] History.
- [x] Download task as pdf or others.
- [x] Mark as done.
- [x] Warn when not done when time reached.

- HTML
- CSS
